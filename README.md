### Игры разума
Пять арифметических игр, построенных по принципу вопрос - ответ. 
При ответе правильно три раза подряд игрок выигрывает. При неправильном ответе игра заканчивается поражением игрока. 
### installation:
```
git clone
make install
sudo npm link
```
